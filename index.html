Julia MINC

Read and write MINC files from Julia, reading and writing .xfm transformation files, Manipulations with volumes

Examples

Read a 3D minc volume, calculate mean value

Minc command:

> mincstats -mean mni_icbm152_t1_tal_nlin_sym_09c.mnc
Mean:              29.61005195

Julia version:

using jminc
using StatsBase

# read volume from minc file, represent it as array of doubles (Float64 in julia world)
icbm=jminc.read_volume("mni_icbm152_t1_tal_nlin_sym_09c.mnc", store=Float64)
@show mean(jminc.array(icbm))

Output mean(jminc.array(icbm)) = 29.61005194874031

Read a 3D volume with real values, and another one with mask labels and show statistics per label

using jminc
using StatsBase

# read T1w scan
icbm=jminc.read_volume("mni_icbm152_t1_tal_nlin_sym_09c.mnc", store=Float64)
# read label mask, represent it as array of bytes 
lab=jminc.read_volume("mni_icbm152_cls_tal_nlin_sym_09c.mnc", store=UInt8)

# calculate image statistics per label
for (i,l) in [1=>"CSF",2=>"GM",3=>"WM"]
    println("$(l):$(mean(jminc.array(icbm)[ jminc.array(lab).==i ]))")
end

Output:

CSF:31.263195070792978
GM:64.88399427312729
WM:84.74593912912998

Apply transformation stored in .xfm file to a minc volume, save result

Create transformation:param2xfm -rotations 30 0 0 rotate.xfm

using jminc

# read T1w scan
icbm=jminc.read_volume("mni_icbm152_t1_tal_nlin_sym_09c.mnc", store=Float64)

# read transformation
tfm=jminc.load_transforms("rotate.xfm")

# apply transformation to the volume
transformed_icbm=jminc.resample_volume(icbm; tfm, order=2, fill=0.0)

# write the resulting volume into file, append history
# the resulting volume will be stored with unsigned short minc data type
# but will preserve floating point vales used fixe-point arithmetics
jminc.save_volume("transformed_icbm.mnc",transformed_icbm, store=UInt16, history="Julia example")

Integrate jacobians per ROI, based on a transformation in .xfm file

using jminc

# read ROI labels
rois = jminc.read_volume("rois.mnc", store=UInt8)

# calculate voxel volume 
start, step, dir_cos = jminc.decompose(jminc.voxel_to_world(rois))
voxel_volume = abs(reduce(*,step,init=1))

# read transformation
tfm = jminc.load_transforms("nonlinear.xfm")

# define sampling parameters for jacobian field
jacobians = jminc.empty_volume_like(lab, store=Float64)

# calculate per-voxel jacobians
jminc.calculate_jacobian!(tfm, jacobians)

# Integrate per ROI
for (i,l) in [1=>"ROI-1",2=>"ROI-2",3=>"ROI-3"]
    println("$(l):$( sum((jminc.array(rois) .== i) .* jminc.array(jac) )*voxel_volume)")
end

Graphic examples, using Makie

Show MNI-ICBM152 template contours with tissue masks overlays

using CairoMakie
using jminc

# read T1w scan
icbm=jminc.read_volume("mni_icbm152_t1_tal_nlin_sym_09c.mnc", store=Float64)
# read label mask, represent it as array of bytes 
lab=jminc.read_volume("mni_icbm152_cls_tal_nlin_sym_09c.mnc", store=UInt8)


fig = Figure()
gc = fig[1, 1] = GridLayout()

jminc.draw_outline_with_labels(gc, icbm, lab, 
    labels=Dict([1=>"CSF",2=>"GM",3=>"WM"]),
    nslices = 5)

resize_to_layout!(fig)

save("mni_icbm152_segmentation.png", fig, px_per_unit = 1)

Show MNI-ICBM152 template contours with GM proability map

using CairoMakie
using jminc

# read T1w scan
anat=jminc.read_volume("mni_icbm152_t1_tal_nlin_sym_09c.mnc",store=Float64)
# read field
gm=jminc.read_volume("mni_icbm152_gm_tal_nlin_sym_09c.mnc",store=Float64)

fig = Figure()
gc = fig[1, 1] = GridLayout()

# set background to transparent
jminc.array(gm)[jminc.array(gm) .< 1e-6] .= NaN

jminc.draw_outline_with_heatmap(gc, anat, gm, 
    heat_limits=(0.0,1.0),cmap=:turbo,
    nslices = 5)

resize_to_layout!(fig)

save("mni_icbm152_gm.png", fig, px_per_unit = 1)
